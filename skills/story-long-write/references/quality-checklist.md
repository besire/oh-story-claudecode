
# 网文质量检查清单

## 一、通用检查

### 章节结构
- [ ] 开头有钩子（不是天气/风景/日常开场）
- [ ] 中段有推进（有事件发生）
- [ ] 局势有变化（读完这章，世界跟之前不一样了）
- [ ] 结尾落在变化上（不是总结）

### 开篇检查（前 300-500 字）
- [ ] 有钩子，能抓住注意力
- [ ] 不从天气/风景/日常开始
- [ ] 主角快速出场
- [ ] 卖点或危机可见

### 章节推进
- [ ] 有核心事件
- [ ] 局势有变化
- [ ] 不是水字数（删掉这章会影响理解吗？不会 = 水了）
- [ ] 推进了主线/关系/设定中的至少一项

### 信息传递
- [ ] 没有大段设定说明文
- [ ] 信息跟着冲突走（通过事件传递设定）
- [ ] 设定量可控（一章不超 3 个新概念）

### 场景检查
- [ ] 场景有目标（人物要什么）
- [ ] 场景有阻碍（什么挡着）
- [ ] 场景有变化（结束后跟之前不同）
- [ ] 人物在做事情，不是在感觉事情
- [ ] 没有可删除的段落

### 章尾
- [ ] 结尾落在变化上
- [ ] 有危机/决定/发现/反转中的至少一个
- [ ] 不是总结式结尾
- [ ] 拉住读者翻下一页

### 语言
- [ ] 没有空洞的抒情段落
- [ ] 没有连续多段同一情绪
- [ ] 对话符合人物身份（不同人说话方式不同）
- [ ] 情绪通过动作落地（不是直接说"他很难过"）

### 连载连续性
- [ ] 没有遗忘之前的承诺/伏笔
- [ ] 没有突然塞入大量新设定
- [ ] 伏笔有推进
- [ ] 故事引擎还在运转

### 水（filler）检测
以下信号出现 = 可能水了：
- 全章对话没有任何新信息
- 同一个情绪写了 3 段以上
- 场景描写超过 500 字但不推进剧情
- 角色回忆之前发生的事但没有任何新视角
- 连续 2 章以上没有冲突

## 二、长篇专项

### 黄金三章检查
- [ ] 第一章前 500 字有钩子？
- [ ] 主角第一章就出场？
- [ ] 第一章有事件（不是纯铺垫）
- [ ] 第二章有升级（矛盾加深）
- [ ] 第三章有追读理由（给出继续看的动力）
- [ ] 前三章有至少 2 个爽点？
- [ ] 世界观没有大段说明文？
- [ ] 每章结尾有悬念？

### 节奏检查
- [ ] 最近 5 章是否有明确进展？
- [ ] 爽点间隔是否超过 5000 字？
- [ ] 有没有连续 2 章以上没有冲突？

### 人物检查
- [ ] 主角行为符合人设？
- [ ] 配角是否有存在感？
- [ ] 反派逼格是否匹配当前阶段？

### 五维评分标准

每个维度 0-100 分，根据评分结果选择精修策略。

#### 维度 1：核心一致度
检查：关键冲突、关键行动、人物动机是否前后一致。

| 问题 | 严重度 | 修复 |
|------|--------|------|
| 人物动机突然改变无铺垫 | critical | 补充动机转变的触发事件 |
| 核心冲突前后不一致 | high | 回溯修改冲突设定 |
| 关键行动与人物性格矛盾 | high | 调整行动或补充解释 |
| 次要矛盾遗忘 | medium | 回收或弱化 |

#### 维度 2：表层重写度
检查：句式与措辞是否足够原创，避免套路化表达。

| 问题 | 严重度 | 修复 |
|------|--------|------|
| 连续复用 12 字以上原文 | high | 重写句式 |
| 大量使用 AI 标志词 | high | 替换为具体描述 |
| 同一句式重复出现 | medium | 变换表达方式 |
| 描写过于文学化 | medium | 改为口语化/动作化 |

#### 维度 3：格式一致度
检查：段落结构、字数分配、开头结尾格式是否统一。

| 问题 | 严重度 | 修复 |
|------|--------|------|
| 段落长度差异过大 | medium | 调整段落划分 |
| 章节字数偏离目标 | medium | 增删内容 |
| 格式混乱（对话/描写不统一）| low | 统一格式 |

#### 维度 4：可读性
检查：是否有啰嗦、AI 腔、空泛总结、套路修辞。

| AI 腔特征 | 如何识别 | 如何修复 |
|-----------|----------|----------|
| 空泛总结 | 「他终于明白了」「一切尽在不言中」 | 删掉，用行动代替 |
| 套路修辞 | 「命运仿佛在和他开玩笑」 | 删掉或换成具体描述 |
| 情绪标签 | 「他感到一阵悲伤」 | 改为行为表现 |
| 大段心理描写 | 连续 3 句以上内心独白 | 压缩为 1 句或转为对话 |

#### 维度 5：逻辑连贯
检查：句间/段间是否通顺，有无设定冲突。

| 问题 | 严重度 | 修复 |
|------|--------|------|
| 设定前后矛盾 | critical | 查找并统一设定 |
| 时间线错误 | high | 标记时间线并修正 |
| 角色信息不一致 | high | 建立角色档案对照 |
| 因果链断裂 | medium | 补充过渡 |

### 精修策略

根据五维评分结果，选择精修策略：

| 主要问题 | 策略 | 说明 |
|----------|------|------|
| 核心一致度低 | rewrite | 围绕核心冲突重写相关段落 |
| 字数超标 | compress | 删减不推动剧情的内容 |
| AI 腔重 | de_ai | 替换禁用词、改写句式 |
| 小问题多 | polish | 打磨语言细节 |

## 三、短篇专项

### 虐爽节奏检查

短篇特有的检查项——虐点和爽点的分布是否合理：

```
理想分布：虐1 → 虐2 → 虐3 → 爽1(小) → 虐4 → 爽2(大)
禁忌分布：虐1 → 虐2 → 虐3 → 虐4 → 虐5（无爽点，读者流失）
禁忌分布：爽1 → 爽2 → 爽3 → 爽4（无铺垫，爽感疲劳）
```

检查规则：
- 每 300 字至少 1 个小冲突
- 虐点间隔不超过全文的 30%
- 最大爽点必须出现在全文的 70-85% 位置
- 结尾必须有情绪落点

### 对话密度检查

| 指标 | 合格标准 | 警戒线 |
|------|----------|--------|
| 对话占全文比 | 45-65% | <30%（太干）或>75%（太水） |
| 每章对话条数 | 12-18条/千字 | <8条（缺交互）或>25条（碎片化） |
| 伤人性对话占比 | 35-45%（虐文）/ 20-30%（爽文） | 虐文<20%（不够痛）/ 爽文>40%（太压抑） |

### 主角冷静度检查（打脸/复仇文专用）

| 检查项 | 标准 |
|--------|------|
| 主角是否有标志性冷静动作？ | 必须有（端水杯/整理西装/转笔等） |
| 主角是否有情绪失控场面？ | 复仇文最多1次，且必须在前30% |
| 反派是否比主角更歇斯底里？ | 必须是——反差是爽感来源 |
| 主角台词是否短于反派？ | 平均短30-50%——越短越有力 |
| 是否有"审判式对话"？ | 至少2处（主角提问→对方自爆） |

### 证据链完整性检查（复仇/打脸文专用）

| 检查项 | 标准 |
|--------|------|
| 证据是否分章释放？ | 至少分3次揭露，不能一次全给 |
| 每个证据是否有铺垫？ | 前文必须埋过线索 |
| 反派是否每次都先得意再被打脸？ | 必须是——先扬后抑 |
| 最终证据是否最致命？ | 最后一个证据要改变全局认知 |
| 是否有"定时炸弹"证据？ | 至少1个主角提前布局的证据 |

### 毒点排查检查

| 检查项 | 标准 | 说明 |
|--------|------|------|
| 主角是否"不爽"到了让读者难受？ | 爽文不爽就是毒 | 金手指效果必须展示得清清楚楚 |
| 主角是否被"莫名其妙"地压制？ | 每次压制必须有目的 | 不能为了拉长篇幅而压 |
| 反派死亡是否太简单/与主角无关？ | 反派结局必须与主角关联 | 完读率杀手 |
| 经济系统是否崩坏？ | 设定要前后一致 | 资源获取速度 vs 价格要合理 |
| 女主/女配是否降智？ | 人设正常可写，老套路别写 | 嘤嘤怪/被抓走/拿了装备不打输出 |
| 读者期待是否被长时间不满足？ | 适时满足或引入新期待 | 连续破坏 = 信任危机 |
| 主角行为读者能否理解？ | 必须能理解、能共鸣、能接受 | 作者屁股必须在主角那边 |
| 氛围感是否被突兀的梗破坏？ | 氛围 > 突兀梗 | 除乐子文外 |
| 结局是否回应了开篇的钩子？ | 必须回应 | 钩子不回收 = 烂尾 |

### 代入感检查

| 检查项 | 标准 |
|--------|------|
| 主角是否主动为自己创造条件？ | 被动人物立不住 |
| 反派是否令人生畏？ | 反派不够强 = 主角赢了没意义 |
| 心理描写是否"润物细无声"？ | 加括号/标记 = 刻意 = 破坏代入 |
| 展示是否优于告知？ | 用行动/选择展现性格，不是描述 |
| 是否"贴着人去写"？ | 行为/语言/思维围绕人格展开 |
| 读者已知信息是否被有效操控？ | 控制信息差制造情绪波动 |

### 震惊效果检查

| 检查项 | 标准 |
|--------|------|
| 震惊是否分层？ | 点震惊→网震惊→深度震惊，不能只写点震惊 |
| 震惊是否阶梯递进？ | 不是一条直线，而是阶梯式上升 |
| 是否用道具递进体现震惊？ | 如：裂痕→满是裂纹→捏爆 |
| 震惊是否有"广度"？ | 不只一个人震惊，关系网都要有反应 |

### 开头冲击力检查

| 检查项 | 标准 |
|--------|------|
| 前50字是否出现冲突/异常？ | 必须有，不能是背景铺垫 |
| 前100字读者是否知道核心矛盾？ | 必须知道 |
| 是否有"钩子句"？ | 至少1句让读者想继续看 |
| 开头情绪强度（1-10）？ | 必须≥7 |

### 结尾力度检查

| 检查项 | 标准 |
|--------|------|
| 最后一句是否是具体的（动作/对话/画面）？ | 必须是，禁止总结/反思/升华 |
| 结尾是否有余韵？ | 读者应该还想往下看/翻回开头重看 |
| 是否有标志性动作/台词收尾？ | 最好有 |
| 结尾情绪强度（1-10）？ | 虐文≥8，爽文≥7，治愈文≥6 |

### 情绪对等释放检查

| 检查项 | 标准 |
|--------|------|
| 反派做了多恶劣的事，主角反击是否有多爽？ | 必须对等——投入多大负面情绪，释放时就要有对应量级的正面释放 |
| 反派结局是否与主角关联？ | 必须关联——反派死亡/落败必须因主角行动导致 |
| 爽点是否到位？ | 该爽不爽到位比毒点还毒 |
| 金手指效果是否展示清楚？ | 必须清清楚楚 |

### 期待感管理检查

| 检查项 | 标准 |
|--------|------|
| 读者读到这里在期待什么？ | 每个位置都要能回答 |
| 期待是否被长时间不满足？ | 适时满足或引入新期待 |
| 满足旧期待时是否引入新期待？ | 保持"两长一短"不断 |
| 核心卖点是否贯穿始终？ | 偏离卖点 = 断崖式下跌 |
| 信息差是否被有效利用？ | 读者知道但角色不知道 = 天然期待 |

### 毒点排查核心

| 毒点类型 | 说明 | 排查方法 |
|----------|------|----------|
| 偏离核心梗 | 冲着核心梗来的读者会觉得"没那味儿" | 每段剧情对照核心梗检查 |
| OOC人设崩塌 | 角色做出与已建立人设不符的行为 | 角色行为是否与标签/动机一致 |
| 反派降智 | 反派变蠢来给主角送赢 | 反派行为是否在其视角说得通 |
| 主角犯错 | 下行情节因主角自身失误导致 | "功都是主角的，锅都是别人的" |
| 情绪不对等 | 反派作恶程度 vs 主角反击爽度 | 虐1点憋屈，报复至少10倍 |
| 题材边界越界 | 写了目标读者群体不想看的元素 | 了解目标读者的共有毒点 |
| 无意义虐主 | 虐没有目的，不为后续爆发蓄力 | 每次下压都要为后续上扬服务 |

**毒点识别方法**：
- 复数读者同样批判某个剧情 = 确认毒点
- "汝之蜜糖，彼之砒霜"：不同读者群体毒点不同，必须明确目标读者
- 题材边界覆盖50%-80%以上，剩余部分读者会容忍

## 毒点排查

> 核心原则：毒点是让读者产生强烈不适、弃书的情节或人物设计。了解毒点才能规避，有时候毒点用得好也能成为特色。

### 一、内容红线

#### 审核风险
- 擦边和三观不正的内容写明目张胆容易出事
- 即使写外国背景也有风险，编辑可能随时保不住
- 有人举报就完了，不要心存侥幸
- 涉政内容更敏感，编辑说了算，"灵活尺度"永远问不出来

#### 女频毒点
- 双洁是铁律：不仅是处男处女，还要求初恋和唯一白月光
- 女主曾经喜欢别的男人在男频可能是抢过来的快感，在女频是完全毒点
- 补救方案：女配和女主其实是同一个人（转世/穿越）

### 二、人设毒点

#### 主角设计
- 主角太弱势：绿茶角色对主角有正面作用但让读者膈应，说明主角偏弱
- 牛逼的主角身边任何角色都能变成正反馈
- 以人设为核心卖点很难写，新手大概率掌握不了
- 主角不能有low的特质（如见美女紧张），可以放配角

#### 绿茶/负面女配
- 立体但让人恶心的角色可以推动剧情
- 但要问自己：这个角色是否有不可替代性？
- 能不用让人恶心的方式推动剧情，就不要用
- 在整体情绪持续向上的文里可以起到压抑情绪的调节作用

#### 角色处理
- 群友建议写死负面角色——但不太建议，显得粗暴
- 建议把负面角色设计成不值得被爱的类型——但也在挑战底线
- 最好的方式是从根源上避免设计这种情况

### 三、情节毒点

#### 常见弃书点
- 金手指出现得太突兀（如写日记突然冒出金手指）
- 主角目标不清楚，大量无效情节
- 战斗没有情绪只有动作描写
- 无意义的时间线跳跃
- 大段世界观说明文

#### 情绪释放不完善
- 读者能感觉到情绪释放不够完善
- 抗忍耐性的读者可能不介意，但严格要求的读者就觉得不适
- 写完要检查：读者期待的爽点是否给到位了

### 四、逆向利用毒点

#### 有意的"毒"可以成为特色
- 主角抽象、毒点雷人、但因为"主角不是真主角"而独辟蹊径
- 个人风格太强烈也是细分赛道
- 前提是：你的"毒"是有目的的设计，不是无意的失误

#### 底线
- 角色可以毒但不能让读者觉得作者也毒
- 情节可以雷但要有逻辑自洽
- 永远不要侮辱读者智商

### 五、女频毒点详解

#### 双洁铁律
- 不仅是处男处女，还要求初恋和唯一白月光
- 女主曾经喜欢别的男人在男频可能是抢过来的快感，在女频是完全毒点
- 男主一开始对女配有好感 = 触碰双洁底线，几乎无解

#### 双洁违规的补救方案
- 唯一解法：女配和女主是同一个人（转世/穿越）
- 女频经典板子：男主和女主幼年相识→因故分离→女配顶替女主身份→男主错认→后续相认+女配身败名裂
- 前提：男主和女配不能有CP行为
- 若已发生CP关系：原男主降为男二，开新真男主，原男主走后悔流
- 逆向板子也可用：原男主喜欢女配忽视女主→女主重生→找真男主

### 六、男频女性角色毒点

#### 为什么女角色容易成毒点
- 动不动分手、嘤嘤怪、关键时刻被反派抓走、拿了装备不打输出
- 大部分作者喜欢拿女主当矛盾点（XX，柳如烟是我的）
- 写半天结果女主是隐藏反派
- 无脑贴男主的女主没人设，纯YY工具人

#### 女角色设计建议
- 不降智的女角色可写，老套路写法不如不写
- 有大量单女主、多女主文非常出色，关键是人设正常
- 后宫文的"翅膀打结"（女主之间亲密接触）是大毒点，两边读者都得罪
- 领主文里随意撮合配角CP也是高风险操作，写好不加分，写烂直接炸

#### 送女毒点
- 有让读者产生"在一起欲望"的女配被送出去 = 读者被背刺
- 女配送去和亲这种理由大概率会炸
- 度很难把控，很多人一看到这种就跑

### 七、系统设定与经济崩坏

#### 经济系统崩坏
- 开局设定灵石多难赚→主角有金手指后资源暴增→为压升级速度随意涨价
- 核心问题：没有以普通人的收入为基准建立物价体系
- 正确做法：设定普通人年收入→设定最贵资源价格→推算主角获取速度与普通人的差距
- 每个等级设一个资源价格上限，特殊物品可以适当超过

#### 战力崩坏
- 主角有几百倍资源还艰难晋升，晋升后同级别的人又跟大白菜一样
- 晋升难度与资源获取不匹配，读者觉得不合理
- 建议：升级后的花销与赚钱速度要跟晋升难度成正比

#### 物价崩坏不是绝症
- 剧情精彩的话差距不太离谱都能接受
- 怕的是作者为了防崩而小心翼翼，主角的钱不敢花
- 设置合理的锚点（普通人年收入）比追求完美经济模型更实际

### 八、同人二设与OOC毒点

#### 同人写作红线
- 严重OOC会逼沉默读者开麦骂人，造成读者大批流失
- 写同人必须反复检查设定，绝对不能有离谱二设
- 修复方法：立刻改，不要发道歉声明，过段时间删评论，当无事发生过

#### 原创同理
- 自创世界观内的角色行为也要符合已建立的人设
- 前期有脑子的反派后期气急败坏硬刚主角 = 人设崩塌
- 智者型角色的失败应该是"机关算尽后无奈认输"或"拼上性命坚持道义"

### 九、展示与情节的毒点

#### 干巴的展示
- 只有视觉描述，没有情节推动 = 白写
- 正确的展示结构：
  1. 铺垫：暴露主角行动的目的
  2. 展示效果+目的是否达成（收获1）
  3. 配角反应对比（收获2）
  4. 主角分析后续可能性（拉期待或新收获）

#### 展示必须包含的要素
- 场景、人物、情节（基本要素）
- 人物目的、人物行动、人物态度
- 情节铺垫、情节反转、情节后续
- 铺垫、期待、爽点最好都在展示情节里存在

#### 反派处理的毒点
- 反派死得太平静 = 读者完读率极速下降
- 反派之死与主角只有一点关系 = 情绪释放不充分
- 合理让反派死去不等于让主角完全抽身
- 必须给读者情绪释放的满足感

### 十、常见毒点速查表

#### 必死毒点（直接弃书）
- 双洁违规（女频）
- 后宫文翅膀打结
- 主角比读者还惨还穷且穿越多年没有改善
- 金手指出现太突兀（如写日记突然冒出系统）
- 严重OOC（同人）
- 擦边/三观不正/涉政被举报

#### 高危毒点（大量流失）
- 送女/女配被撮合CP
- 核心梗偏离（挂羊头卖狗肉）
- 经济/战力崩坏到离谱
- 主角目标不清、大量无效情节
- 大段世界观说明文
- 绿茶女配没有不可替代性

#### 轻度毒点（部分读者介意）
- 心理活动用（）框起来（影响阅读流畅度）
- 章节名起得太随意（影响追读）
- 无意义的时间线跳跃
- 战斗只有动作没有情绪
- 情绪释放不完善（要求严格的读者觉得不适）

### 十一、反派洗白的毒点

#### 洗白的底线
- 不能掩盖杀害族人至亲的事实——这种罪行读者不会接受任何理由
- 洗白只能从角色的其他正面方面入手，但如果正面戏份太少就没有可洗的
- 强行洗白比不洗白更毒：读者会认为作者三观有问题

#### 洗白的正确方式
- 洗白需要足够的前期铺垫：在角色"做坏事"时就要埋下可以理解的动机
- 角色的正面行为要有足够的戏份支撑
- 洗白不能洗掉核心罪行，而是让读者理解"为什么会走到这一步"
- 最好的洗白是让读者自己产生同情，而不是作者按着头让读者接受
